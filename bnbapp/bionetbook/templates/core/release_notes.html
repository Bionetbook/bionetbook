Release 0.2 August 26 2013

New Features:

  • Protocol layout: Lets user view protocol as a diagram or text. Comments can be turned off for easy reading
  • Protocol permissions: You can choose to share your protocols with your organization or with everyone.
  • Protocol compare: Protocols display verb differences and can toggle on/off specific differences amounts and units of reagents, machine and thermocycler settings, etc.

Fixes:

  • Duration calculation: Actions, Steps and Protocols now have duration calculation with a time range ( minimum time - maximum time).
  • Protocol layout: Protocols can be rendered and no longer fail.
  • Dashboard shows all users' protocols.

Known Issues:

  • Login: Users can not create accounts, this will be fixed in a further release
  • Dashboard: Users can't edit personal information, this will be fixed in a further release
  • Protocol entry: We are aware that this process is cumbersome. The next release will improve protocol entry significantly.:
  • Protocol search: There is no search in this release.
  • Protocol compare: Comparing protocols can only be done on cloned protocols. In future versions there will release more comprehensive comparison tools.
  • Protocol compare: Only two protocols can be compared. This will be extended to more in future release.
  • Protocol duration: Duration only accounts for the net time and indicates the estimated amount of time for executing the protocol. In some cases there will be a range, its reflected from the protocol. In future releases duration will account for gross time which includes in-between times like setting up the tubes in a centrifuge, labeling tubes etc.

Next Release Targets

  • Protocol scheduling: You will be able to sync your personal calendar with a protocol schedule.
  • Workflows: Combine several protocols into a workflow.
  • Experiments: Combine several workflows into an experiment.
  • Schedules: Dynamically create schedules for your experiments.
  • Protocol searches: We will add protocol searching tools that integrate PMID resources.
